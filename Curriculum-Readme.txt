说明:
    release 版本是签名版本（正规发布版本），如果有的有些手机不能安装，或出现安装错误，则选择debug（未签名）版本。

Version 2.0
1、增加一个广播监听器，监听时间变化，每分钟刷新一次当天课程的显示颜色。
2、删除每60秒执行一次的线程。
3、消除错误：每天早上第4节课下课后，当天课程的显示颜色会变成当天的课程全部上完的错误。

Version 2.1
1、定置新图标（取代默认图标）。

Version 2.2
1、进入课程设置以后，点击班名称，可以修改班级名称，并且保存在手机里，下次运行程序时读出。
2、在作息时间表里，判断当前时间在哪个时间区域内，则将这个时间区域对应的TextView的背景色变为设定的颜色。

Version 2.3
1、修复错误：在“作息时间表”页面，最后一条时间记录永远不会被点亮的错误。
2、作息时间表在周六和周日不更新颜色。
3、课程为空时不改变颜色。
4、增加当天课程的颜色说明。

Version 2.3.1
1、增加DataInit类，存放初始化数据。发布版本不输出LOG。
